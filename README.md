# Linux File Manipulation Commands
## mkdir

```bash
mkdir hw_files

ls
```
Output:
```text
Desktop    Downloads  Music     Public     Videos
Documents  hw_files   Pictures  Templates
```


## cd and pwd

```bash
cd hw_files
pwd
```

Output:
```text
/home/eugene/hw_files
```
